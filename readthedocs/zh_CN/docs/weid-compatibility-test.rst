通过 yum 命令下载安装的Open JDK
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

不兼容。

.. list-table::
   :header-rows: 1
   :widths: 30 50

   * - yum open JDK版本号
     - 测试结果
   * - 1.8.0_232
     - 验证失败❌
   * - 11.0.5
     - 验证失败❌

不兼容的原因：由于CentOS的yum仓库的OpenJDK缺少JCE(Java Cryptography
Extension)
